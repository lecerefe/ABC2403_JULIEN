# Exercice de calcul de la moyenne de 2 nombres
```
VARIABLES
premiereNombreEntier est un nombre entier
VARIABLES
deuxiemeNombreEntier est un nombre entier
VARIABLES
moyenneDesDeuxNombre est un nombre reel

TRAITEMENT
ECRIRE "Saissisez un premier nombre" <--- premierNombreEntier
LIRE premierNombreEntier
ECRIRE "Saissisez un Deuxieme nombre" <--- deuximeNombreEntier
LIRE deuxiemeNombreEntier
LIRE premierNombreEntier + deuxiemeNombreEntier / 2

moyenneDesDeuxNombre <-- (premierNombreEntier + deuxiemeNombreEntier) / 2

RESULTA 

ECRIRE "La moyenne de " ,premierNombreEntier, "et" , deuxiemeNombreEntier , "est" , moyenneDesDeuxNombre, "."
// "La moyenne de 12 et 81 est 46,5"
